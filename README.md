# voice_recognizer

Голосовой помощник для Windows: распознаёт русскую речь с микрофона и выполняет действия на ПК (управление плеером/видео, запуск/закрытие приложений, переключение окон, автоклики в интерфейсе). В части сценариев (например, “пригласи друзей”) использует OCR: делает скриншот области экрана и ищет нужные слова через Tesseract.

## Возможности

- **Распознавание речи**: `SpeechRecognition` + `recognize_google(..., language="ru-RU")`.
- **Управление клавиатурой/мышью**: `pyautogui` (нажатия клавиш, клики по координатам).
- **Фокусировка окон**: `win32gui` (поиск окна по части заголовка и активация).
- **Озвучка**: `pyttsx3` (попытка выбрать голос “Aleksandr”).
- **OCR по экрану**: `pyscreenshot`/`opencv-python` + `pytesseract`.

## Быстрый старт

### 1) Установка зависимостей

```bash
python -m pip install -r requirements.txt
```

Часто дополнительно требуется:

- **Tesseract OCR**: установите Tesseract и убедитесь, что `tesseract.exe` доступен в `PATH`.
  - Если `pytesseract` не видит бинарник, можно указать путь в коде (пример ниже).
- **PyAudio**: на Windows иногда не ставится из исходников. Если `pip install PyAudio` падает, поставьте подходящий wheel под вашу версию Python или установите через `pipwin`.

Пример фикса для Tesseract (если нужно):

```python
import pytesseract
pytesseract.pytesseract.tesseract_cmd = r"C:\Program Files\Tesseract-OCR\tesseract.exe"
```

### 2) Запуск

```bash
python recognize.py
```

После старта скрипт произнесёт фразу и начнёт бесконечно слушать микрофон.

## Настройка

### Микрофон (важно)

В `recognize.py` используется `sr.Microphone(device_index=1)`. На вашем ПК индекс может быть другим.

Чтобы узнать доступные устройства, временно раскомментируйте блок в начале `recognize.py`:

```python
for index, name in enumerate(sr.Microphone.list_microphone_names()):
    print(index, name)
```

Затем подставьте корректный `device_index`.

### Пути к приложениям

В коде встречаются жёстко заданные пути, например:

- Steam: `C:\Program Files (x86)\Steam\steam.exe`
- Chrome: `C:\Program Files\Google\Chrome\Application\chrome.exe`

При необходимости отредактируйте их под вашу установку.

### Координаты кликов/области экрана

Часть действий завязана на относительные координаты (в процентах от экрана), часть — на конкретные области. Скрипт `coords.py` помогает узнать координаты курсора.

## Примеры голосовых команд

Команды распознаются в нижнем регистре и ищутся через `in` (частичное совпадение). Примеры:

- **Видео/плеер**: “пауза”, “старт”, “вперёд”, “назад”, “разверни видео”, “выключи звук”.
- **Приложения**: “включи стим”, “выключи стим”, “включи браузер”, “закрой браузер”.
- **CS:GO**: “включи контру”, “выключи контру”, “разверни контру”, “начни поиск”, “останови поиск”, “прими игру”, “пригласи друзей”.
- **Система**: “выключи компьютер”, “перезагрузи компьютер”.
- **Остановка**: “выключи себя” (пытается завершить `recognize.exe`).

## Сборка в .exe

В проекте есть `recognize.spec`. Для сборки:

```bash
pyinstaller recognize.spec
```

Результат появится в стандартных папках `dist/` и `build/` (создаются `pyinstaller`).

## Частые проблемы

- **`OSError: [Errno -9996] Invalid input device (no default output device)` / микрофон не слушает**
  - Проверьте `device_index`, права доступа к микрофону и устройство ввода по умолчанию.
- **`pytesseract.pytesseract.TesseractNotFoundError`**
  - Установите Tesseract и добавьте в `PATH` или задайте `tesseract_cmd`.
- **`pip install PyAudio` не ставится**
  - Используйте wheel под вашу версию Python или альтернативный способ установки (например, `pipwin`).

## Безопасность

Скрипт выполняет потенциально опасные действия (закрытие процессов, выключение/перезагрузка ПК, клики и нажатия клавиш). Используйте осознанно и не запускайте на машине, где это может навредить.
